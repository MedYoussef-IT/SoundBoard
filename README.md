SoundBoardMobile is a lightweight Android application that provides multiple soundboards organized into categories.

Users can quickly play sounds for entertainment purposes. The app focuses on fast interaction, simplicity, and clean UI design.

## Features
- Multiple soundboard categories
- Fast sound playback
- Simple and clean interface

## Technologies Used
- Java
- Android SDK

## Project Status
Completed

## Purpose
Created to practice media handling and fast UI interactions in Android applications.
